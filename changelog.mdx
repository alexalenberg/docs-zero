</Update>

<Update label="March 2025">
- [LOCAL](/guides/reporting/local-generation) generation (Actual, Forecast) for metering is added
- [LOCAL](/guides/reporting/local-generation) generation for scheduling and cost reporting is added
- Support for enhanced [Tariff settings](/guides/connect/settings-page) page is added
- [Test (sandbox) mode](/guides/system/test-mode) is now free to use with any subscription
</Update>

<Update label="February 2025">
- Lifecycle webhooks now include [tariff.expiring](/guides/connect/lifecycle#event-types)

- [Price limit](/guides/scheduling/limit) based scheduling released
- [Energy amount](/guides/scheduling/amount) based scheduling now supports up to 1,000 devices
- New [tariff rates audit](/api-reference/zero/tariffs/audit-tariff-rates) API endpoint released
</Update>

<Update label="January 2025">
- Postal addresses are now automatically geocoded in Connect
- [Gulf (NL)](https://gulfgasandpower.nl) integration released
- Connect now supports multiple contracts per address
- Connect now handles missing electricity contracts gracefully
- Connect summary now shows [tariff connection type](/api-reference/zero/connect/tariff-summary#code)
</Update>
