## Automatic mapping

When you [create a location](/api-reference/zero/locations/create-a-location) or [a new token for Connect](/api-reference/zero/connect/create-a-connect-token), we ask you to provide the full postal address of your customer's location. We use this information to automatically map it to their market rate region. Within Flatpeak, this region is represented by a `market_rates_source` ID that you can find via the [audit tariff rates](/api-reference/zero/tariffs/audit-tariff-rates) endpoint.

```json example truncated audit object {5}
{
  "id": "tra_65e42cdd73e9f861edda35d7",
  "object": "tariff_rate_audit",
  "market_sources": [
    "mrs_65792b4df61b16685ebd2e78"
  ]
}
```
